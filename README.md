# Cy-log
Cy-log is a web based application that allows a user to keep a scientific journal filled with entries for each day.

# Approach Taken
I used a React front-end that can perform all CRUD actions on a Rails api back-end.

# Technology Used
- React
- node.js
- html
- css
- ruby
- rails
